# 🔐 Secure File Sharing System

A secure desktop application developed using Python for secure file upload, encryption, decryption, and management.

---

## 🚀 Features

- User Registration
- User Login Authentication
- Secure Dashboard
- File Upload
- File Encryption
- File Decryption
- File Download
- File Deletion
- Secure File Management

---

## 🛠 Technologies Used

- Python
- Tkinter
- SQLite
- Cryptography (Fernet)
- File Handling
- OS Module

---

## 🔒 Security Features

- Fernet Symmetric Encryption
- User Authentication
- Secure File Storage
- Encryption Key Generation

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/singhshobhit-source/Secure-File-Sharing-System.git
```

Install dependencies:

```bash
pip install cryptography
pip install tkinterdnd2
```

Run the project:

```bash
python main.py
```
